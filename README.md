# Meu Projeto Spring Boot

## 📌 Sobre o projeto

Projeto desenvolvido para estudo e prática de desenvolvimento backend utilizando Java e Spring Boot.

A aplicação tem como objetivo aplicar conceitos de criação de APIs REST, organização em camadas, persistência de dados e boas práticas de desenvolvimento.

## 🚀 Tecnologias utilizadas

- Java
- Spring Boot
- Maven
- Spring Data JPA
- Banco de dados (informar o banco utilizado)
- Postman para testes de API
- Git e GitHub

## ⚙️ Funcionalidades

- Cadastro de registros
- Consulta de dados
- Atualização de informações
- Exclusão de registros
- API REST com operações CRUD
